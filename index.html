Sejarah Kementerian Agama

1946

Pembentukan Kementerian Agama

Kementerian Agama Republik Indonesia dibentuk pada tanggal 3 Januari 1946 berdasarkan Peraturan Pemerintah No. 2 Tahun 1946. Kementerian ini didirikan untuk mengurus urusan keagamaan di Indonesia yang baru merdeka.

1950

Pembentukan KUA

Kantor Urusan Agama (KUA) dibentuk sebagai unit pelaksana teknis di tingkat kecamatan untuk menangani urusan pernikahan, perceraian, rujuk, dan administrasi keagamaan lainnya bagi umat Islam.

1975

UU Perkawinan No. 1 Tahun 1974

Undang-Undang Perkawinan disahkan yang memperkuat peran KUA dalam pencatatan pernikahan. KUA menjadi satu-satunya lembaga yang berwenang mencatat pernikahan umat Islam di Indonesia.

2000an

Era Reformasi dan Modernisasi

KUA mulai mengalami modernisasi dengan penerapan sistem informasi dan digitalisasi layanan. Pelayanan menjadi lebih transparan dan efisien dengan dukungan teknologi.

2020an

Transformasi Digital

KUA mengalami transformasi digital penuh dengan sistem online, aplikasi mobile, dan pelayanan berbasis web. Masyarakat dapat mengakses informasi dan layanan KUA secara digital seperti sistem ini.

⚙️ Setup Firebase Config

📋 Cara Setup Firebase:

  1. Buka Firebase Console
  2. Buat project baru atau pilih project yang ada
  3. Pilih "Web App" dan daftarkan aplikasi
  4. Copy API Key, Project ID, dan App ID ke form di atas
  5. Di Firebase Console, aktifkan Firestore Database
  6. Set rules Firestore ke mode test (allow read, write)
  7. Klik "Inisialisasi Firebase" untuk menghubungkan
